# Trabajo práctico nº1. Curriculum Web - Ramallo Martín - https://tincho1980.github.io/cv-ramallo-martin/

Sitio estático tipo curriculum / portfolio profesional para **Martín Ramallo** (Full Stack & Tech Lead). Incluye secciones de presentación, stack tecnológico, experiencia, proyectos y contacto.

Para que esa URL funcione, publicá el sitio con **GitHub Pages** (Settings → Pages) desde la rama que uses, o reemplazá el enlace del título por la URL de otro hosting (Netlify, Vercel, etc.).

## Contenido e idiomas

- **Español:** raíz del sitio — `index.html`
- **Inglés:** `en/index.html`
- Conmutador de idioma con preferencia guardada en `localStorage` y redirección al cargar.

## Tecnologías

- HTML5, CSS3 y JavaScript (vanilla)
- Fuentes: Google Fonts (Space Mono, Fraunces)
- Iconos: Font Awesome (CDN)

## Estructura principal

| Ruta | Descripción |
|------|-------------|
| `index.html` | Página principal (ES) |
| `en/index.html` | Versión en inglés |
| `css/styles.css` | Estilos |
| `js/` | Scripts (loader, matrix, navegación, etc.) |
| `assets/` | Imágenes, iconos de stack y mocks de proyectos |

## Cómo verlo en local

Abrí `index.html` en el navegador o serví la carpeta del proyecto con un servidor estático (por ejemplo Live Server en VS Code / Cursor).
